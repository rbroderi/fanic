FAQ

FANIC is a site for fan-created comics. Creators can upload comics and readers can browse and read them in the browser.

What is this website for?

This site is for fan created comics, including uploading, organizing, and reading fan works.

What content is restricted?

Photorealistic content is not allowed to be uploaded. Any explicit content must be marked as explicit when submitted.

Do I need an account to upload?

Yes. You must login to an account before uploading any comic.

How do explicit-view permissions work?

To view explicit content, you must opt in on the profile page.

Please make sure your uploads follow these rules so content can be safely moderated and correctly surfaced to readers.
